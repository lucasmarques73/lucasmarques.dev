## Sobre o Projeto

Plataforma web desenvolvida para conectar pessoas que desejam adotar animais com ONGs, protetores independentes e feiras de adoção. O objetivo é facilitar o processo de adoção responsável de pets e promover o bem-estar animal.

A aplicação permite que os usuários busquem pets disponíveis para adoção em sua região e também encontrem informações sobre feiras de adoção próximas.

## Principais Tecnologias

- **React** - Biblioteca JavaScript para interfaces
- **Node.js** - Ambiente de execução JavaScript
- **MongoDB** - Banco de dados NoSQL
- **Express** - Framework web para Node.js
- **API de Geolocalização** - Para busca por localização
